# Caveman 优化建议

## 项目介绍

**caveman** 是一个 Claude Code / Codex 插件，通过让 AI 用"穴居人语言"回复，将输出 token 减少约 **65-75%**，同时保持技术准确性。

### 为什么火？
- 🎯 **极致压缩**: 减少 65-75% token 使用量
- 📱 **多平台支持**: Claude Code, Codex, Gemini CLI, Cursor, OpenCode
- 🧪 **有 Benchmark**: 经过 evals 验证
- 🆓 **免费开源**: MIT 许可证
- 💡 **创意有趣**: 病毒式传播的 meme 项目
